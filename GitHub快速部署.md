# ⚡ GitHub Pages 快速部署（3分钟）

## 🎯 只需5步

### 1️⃣ 访问 GitHub
```
https://github.com
```
- 如果没账号，点击 "Sign up" 注册
- 如果有账号，点击 "Sign in" 登录

---

### 2️⃣ 创建仓库
1. 点击右上角 **"+"** → **"New repository"**
2. 仓库名：`fund-portfolio`
3. 选择 **"Public"**（公开）
4. 勾选 **"Add a README file"**
5. 点击 **"Create repository"**

---

### 3️⃣ 上传文件
1. 点击 **"Add file"** → **"Upload files"**
2. 把 `index.html` 拖到页面上
3. 点击 **"Commit changes"**

---

### 4️⃣ 启用 Pages
1. 点击 **"Settings"**（设置）
2. 左侧菜单点击 **"Pages"**
3. **Branch** 选择 **"main"**
4. 点击 **"Save"**

---

### 5️⃣ 获取网址
等待 1-2 分钟，刷新页面，你会看到：
```
Your site is live at https://你的用户名.github.io/fund-portfolio/
```

**完成！** 🎉

## 🔐 多设备同步

### 记住用户ID：
1. 打开网站，按 F12
2. 输入：`console.log(userId)`
3. 记录显示的ID

### 在其他设备使用：
1. 打开网站，按 F12
2. 输入：`localStorage.setItem('userId', '你的ID'); location.reload();`
