# 智能定投系统 - 拆分版

> 一个专业的基金定投管理系统，支持PC端和移动端独立访问

## 🌟 特性

- 📱 **移动端优化** - 专为手机设计的精简界面
- 💻 **PC端完整** - 功能齐全的桌面版本
- 🔄 **自动适配** - 根据设备自动跳转
- ☁️ **云端同步** - 数据实时同步到Supabase
- 📊 **实时数据** - 全球市场指数和基金净值
- 🎨 **现代设计** - 简洁美观的用户界面

## 📱 移动端功能

- ✅ 全球市场指数看板（纳斯达克、标普、沪深300等）
- ✅ 持仓看板（市值、收益、基金数统计）
- ✅ 详细持仓列表（按板块分组，卡片式展示）
- ✅ 用户ID管理
- ✅ 云端数据同步

## 💻 PC端功能

- ✅ 所有移动端功能
- ✅ 基金录入和编辑
- ✅ 板块管理
- ✅ 再平衡分析
- ✅ 复利推演
- ✅ 实时预估
- ✅ 图表展示
- ✅ 数据导入导出

## 🚀 快速开始

### 1. 准备文件

```bash
# 重命名入口文件
index-new.html → index.html

# 临时PC端方案
原index.html → pc.html
```

### 2. 部署到GitHub Pages

#### 使用GitHub Desktop（推荐）

1. 下载安装 [GitHub Desktop](https://desktop.github.com/)
2. 创建新仓库
3. 添加所有文件
4. 提交并发布
5. 启用GitHub Pages

详细步骤见 [部署说明.md](部署说明.md)

#### 使用网页上传

1. 登录 [GitHub](https://github.com)
2. 创建新仓库
3. 上传所有文件
4. 在Settings → Pages中启用

### 3. 访问

```
https://你的用户名.github.io/仓库名/
```

- 手机访问：自动跳转到移动端
- 电脑访问：自动跳转到PC端

