## 🚀 5分钟部署到 Vercel

### 步骤1：访问 Vercel
```
https://vercel.com
```

### 步骤2：登录
- 推荐：使用 GitHub 登录
- 或者：使用邮箱注册

### 步骤3：上传文件
1. 点击 "Add New..." → "Project"
2. 滚动到底部找到 "Or, deploy without Git"
3. 拖拽 `index.html` 文件
4. 或点击 "Browse" 选择文件

### 步骤4：部署
1. 项目名称：`fund-portfolio`
2. 点击 "Deploy"
3. 等待 30 秒

### 步骤5：完成
- 复制你的网址：`https://fund-portfolio-xxx.vercel.app`
- 收藏网址
- 开始使用！

## 📱 在手机上使用

### iOS：
1. Safari 打开网址
2. 点击分享 → "添加到主屏幕"

### Android：
1. Chrome 打开网址
2. 菜单 → "添加到主屏幕"

---

## 🔐 多设备同步

### 记住你的用户ID：
1. 打开网站
2. 按 F12
3. 输入：`console.log(userId)`
4. 记录显示的ID

### 在其他设备使用：
1. 打开网站
2. 按 F12
3. 输入：`localStorage.setItem('userId', '你的ID'); location.reload();`
