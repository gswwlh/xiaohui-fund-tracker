## 🎯 如何在多设备间同步数据

### 场景：你在电脑上添加了数据，想在手机上查看

#### 方法1：使用用户ID同步（推荐）⭐

**在电脑上：**
1. 打开网站
2. 点击顶部的 **"🔑 用户ID"** 按钮
3. 在弹窗中点击 **"复制"** 按钮
4. 用户ID已复制到剪贴板
5. 通过微信/邮件等方式发送给自己

**在手机上：**
1. 打开网站
2. 点击顶部的 **"🔑 用户ID"** 按钮
3. 在 "设置新的用户ID" 输入框中粘贴刚才复制的ID
4. 点击 **"✅ 设置ID"** 按钮
5. 确认后，数据自动从云端加载
6. 完成！🎉

#### 方法2：使用备份/恢复功能

**在电脑上：**
1. 点击 **"⬇️ 备份配置"** 按钮
2. 下载 JSON 文件
3. 发送到手机

**在手机上：**
1. 点击 **"⬆️ 恢复配置"** 按钮
2. 选择 JSON 文件
3. 数据恢复完成

### 弹窗内容：

```
┌─────────────────────────────────────┐
│  🔑 用户ID管理                      │
├─────────────────────────────────────┤
│                                     │
│  当前用户ID                         │
│  ┌─────────────────────┬─────┐     │
│  │ user_xxxxx_xxxxx    │复制 │     │
│  └─────────────────────┴─────┘     │
│  💡 在其他设备上使用相同的用户ID，  │
│     即可同步数据                    │
│                                     │
│  设置新的用户ID                     │
│  ┌─────────────────────────────┐   │
│  │ 粘贴用户ID...               │   │
│  └─────────────────────────────┘   │
│  ⚠️ 设置新ID后，将加载该ID对应的   │
│     数据                            │
│                                     │
│  [ 🔄 重置ID ] [ ✅ 设置ID ] [取消] │
└─────────────────────────────────────┘
```

### 按钮说明：

1. **复制按钮**
   - 一键复制当前用户ID到剪贴板
   - 方便分享到其他设备

2. **✅ 设置ID**
   - 输入新的用户ID
   - 点击后立即切换并加载数据

3. **🔄 重置ID**
   - 生成全新的用户ID
   - 原数据保留在云端（需要旧ID访问）
   - 适合重新开始或创建新账户

4. **取消**
   - 关闭弹窗，不做任何更改

## 💡 使用场景

### 场景1：在新设备上使用

1. 在新设备打开网站
2. 点击 "🔑 用户ID"
3. 输入你的用户ID
4. 数据自动同步

### 场景2：多人共享数据

1. 一个人创建数据
2. 复制用户ID分享给其他人
3. 其他人设置相同的ID
4. 所有人看到同一份数据

### 场景3：创建多个账户

1. 点击 "🔄 重置ID"
2. 生成新的用户ID
3. 开始使用新账户
4. 需要切换回旧账户时，输入旧ID即可

---

## ⚠️ 重要提示

### 1. 用户ID很重要
- 这是访问你数据的唯一凭证
- 建议记录下来或截图保存
- 丢失ID将无法访问数据

### 2. 重置ID的影响
- 重置后会生成新ID
- 旧ID的数据仍在云端
- 需要旧ID才能访问旧数据
- 建议重置前先备份

### 3. 数据安全
- 只有知道用户ID的人才能访问数据
- 不要随意分享你的用户ID
- 定期使用"备份配置"功能
